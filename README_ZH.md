## 功能特性

- **双模式支持**：同时支持 Figma Slides 和 Figma Design
- **像素级完美**：精确保留每个设计细节
- **灵活尺寸配置**：自动配置或手动设置幻灯片尺寸，提供四种调整模式（Fit、Fill、Stretch、Original）
- **实时预览**：导出前导航和预览多个页面
- **质量控制**：可调节导出质量，支持 1x 到 4x
- **设备兼容**：基于位图的幻灯片确保在所有设备和平台上保持一致的视觉效果

## 使用方法

1. 打开你的 Figma Slides 文件或 Figma Design
2. 启动 PixelPerfect PPT 插件
3. 配置导出设置：
   - 选择导出模式（全部或选中）
   - 设置质量等级
   - 配置幻灯片尺寸（原始尺寸或自定义尺寸及调整模式）
4. 使用分页控件预览幻灯片
5. 点击「Export to PPTX」并监控进度

## 工作原理

插件将你的设计导出为高质量位图并嵌入 PowerPoint 格式。虽然导出的幻灯片无法在 PowerPoint 中编辑，但这确保了完美的视觉保真度——所有字体、效果和设计细节都能精确保留。基于位图的幻灯片确保在所有设备和平台上都能保持一致的视觉效果，非常适合分发或在其他设备上演示。

## 系统要求

- Figma Slides 文件或 Figma Design
- Figma 桌面应用或浏览器
