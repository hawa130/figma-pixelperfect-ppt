# PixelPerfect PPT

Export your Figma Slides to PowerPoint presentations with pixel-perfect image quality.

## Features

- **High-fidelity export** - Uses Figma's native export to generate pixel-perfect bitmap images
- **Flexible options** - Export all slides or selected slides only
- **Quality control** - Choose from 1x to 4x export quality levels
- **Automatic sorting** - Slides sorted by page numbers automatically
- **Privacy first** - Runs entirely locally with no network access

## How to Use

1. Open your Figma Slides file
2. Launch the plugin
3. Choose export mode (all slides or selected)
4. Set quality level
5. Click "Export to PPTX"

Perfect for designers who need to share Figma Slides in PowerPoint while maintaining perfect visual fidelity.

# PixelPerfect PPT

将你的 Figma Slides 导出为 PowerPoint 演示文稿，实现像素级完美还原。

## 功能特点

- **高保真导出** - 使用 Figma 原生导出生成像素级完美位图
- **灵活选项** - 导出全部幻灯片或仅导出选中的幻灯片
- **质量控制** - 支持 1x 到 4x 多种导出质量等级
- **自动排序** - 根据页码自动排序幻灯片
- **隐私优先** - 完全本地运行，不访问网络

## 使用方法

1. 打开你的 Figma Slides 文件
2. 启动插件
3. 选择导出模式（全部或选中）
4. 设置质量等级
5. 点击"Export to PPTX"

非常适合需要在 PowerPoint 中分享 Figma Slides 并保持完美视觉保真度的设计师。
