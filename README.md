## Features

- **Dual Mode Support**: Works with both Figma Slides and Figma Design
- **Pixel-Perfect Quality**: Preserves every design detail exactly as created
- **Flexible Dimensions**: Automatically configure or manually set slide dimensions with four resize modes (Fit, Fill, Stretch, Original)
- **Real-Time Preview**: Navigate and preview multiple pages before exporting
- **Quality Control**: Adjustable export quality from 1x to 4x
- **Device Compatibility**: Bitmap-based slides ensure consistent appearance across all devices and platforms

## Usage

1. Open your Figma Slides file or Figma Design
2. Launch the PixelPerfect PPT plugin
3. Configure export settings:
   - Choose export mode (all slides or selected)
   - Set quality level
   - Configure slide dimensions (original or custom with resize modes)
4. Preview your slides using pagination controls
5. Click "Export to PPTX" and monitor progress

## How It Works

The plugin exports your designs as high-quality bitmap images embedded in PowerPoint format. While the exported slides cannot be edited in PowerPoint, this ensures perfect visual fidelity—every font, effect, and design detail is preserved exactly as designed. Bitmap-based slides ensure consistent appearance across all devices and platforms, making them ideal for distribution and presentation on any device.
