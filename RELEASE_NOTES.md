# Release Notes

## Version 1.2.0

**Release Date:** 2025/11/23

### What's New

- Custom Dimensions: Set custom width and height for PPTX slides. Images will be automatically cropped to fit the specified dimensions.
- Dynamic UI Resizing: The plugin UI now automatically adjusts its size based on content.

## Version 1.1.0

**Release Date:** 2025/11/20

### What's New

- Export Progress Indicator: See real-time progress as slides are exported.
- Cancel Export: Cancel an export in progress. The export will stop after the current slide finishes processing.
